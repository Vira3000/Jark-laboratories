# JARK Laboratories

JARK Laboratories is an independent online research and innovation laboratory founded by Mr. Pruden Jark Gideon.
The laboratory is dedicated to exploring advanced technologies, solving real-world challenges, and developing innovative solutions that contribute to a better future.

## Project Overview

This repository contains the website for JARK Laboratories, built as a static landing page with focus on research areas, mission, and contact.

## Deployment

The site is designed to deploy on GitHub Pages.

### Publish with GitHub Pages

1. Push this repository to GitHub.
2. In GitHub repository settings, enable Pages from the `main` branch and the root folder.
3. Visit the published site URL shown in the repository settings.

### GitHub Actions

This repository also includes a workflow for automatic Pages deployment using GitHub Actions.

## Files

- `index.html` — site content and structure
- `styles.css` — page styling and brand colors
- `.github/workflows/pages.yml` — GitHub Actions workflow for Pages deployment
